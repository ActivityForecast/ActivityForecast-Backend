## 배포 상태
- Production 배포: 2025-10-28 활성화
- Docker Compose 운영 환경 적용
- 데이터베이스 영속성 테스트 진행 중

## 주요 기능
- Spring Boot 3.5.7 기반 REST API
- JWT 인증 시스템
- AI 기반 활동 추천
- 크루 기능
